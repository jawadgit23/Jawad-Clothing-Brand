 Project Overview

Jawad Clothing Brand is a frontend-focused e-commerce UI project that simulates a modern fashion brand website.

The goal of this project is to create a visually appealing, user-friendly shopping experience using only core web technologies — without relying on frameworks.

It demonstrates strong understanding of:

Layout structuring
Visual hierarchy
Branding design
Responsive UI development
